CONTROL DETAILS:
================

🎚️ GAIN CONTROL
   Purpose: Final output level adjustment
   Default: 0.0 dB
   Use: Compensate for level changes after processing

🎚️ BLEND CONTROL  
   Purpose: Mix between original and processed signal
   Default: 1.0 (fully processed)
   Use: Subtle integration of headphone correction

🎚️ CROSSFEED CONTROL
   Purpose: Reduce stereo separation for headphones
   Default: 0.15
   Use: Make headphone listening more natural

⚡ ULTRA LOW LATENCY
   Purpose: Bypass compressor and crossfeed for minimal delay
   Default: Off
   Use: Enable for real-time monitoring

🎧 MODEL SELECTION
   Purpose: Choose headphone EQ curve
   Models: 8 different headphone response profiles
   Use: Match your specific headphone model

📡 CODEC COMPENSATION
   Purpose: Compensate for wireless codec artifacts
   Options: Off, SBC/AAC, aptX, LDAC
   Use: Enhance wireless headphone audio quality

TECHNICAL FEATURES:
==================
• 8-band parametric EQ per channel (20Hz - 20kHz)
• Real-time biquad filter processing
• Crossfeed algorithm with configurable delay (0.25ms)
• Dynamic range compression (bypassed in ULL mode)
• Thread-safe parameter automation
• Sample rates: All standard rates supported
• Formats: VST3 + CLAP compatible

The plugin provides professional-grade headphone correction and enhancement, 
with an intuitive interface for real-time parameter adjustment.
